# Codex作業指示：03_編成投稿フォーム

## 1. 目的

`saisoku` に、高難易度向け編成共有機能の一部として **編成投稿フォーム** を追加・整備する。

編成投稿フォームは、団員が高難易度クエストやレヴァンス系周回向けの編成を投稿するための入力画面である。

このフォームでは、毎回ゼロから入力するのではなく、プリセットを選択して初期値を反映し、変更したい箇所だけ編集して投稿できることを重視する。

---

## 2. 前提として参照するドキュメント

作業前に、必ず以下を確認すること。

```text
docs/requirements/01_overall_policy.md
```

---

## 3. 基本方針

編成投稿フォームは、以下の方針で作成する。

```text
最低限の入力だけで投稿できる
プリセットから初期値を反映できる
反映後に自由に編集できる
画像投稿中心にはしない
キャラ・召喚石・武器は候補リストから選択する
候補にないものは自由入力できる
高難度攻略用と周回・武器集め用で表示項目を切り替える
```

---

## 4. 投稿開始時の流れ

投稿開始時は、以下の流れを想定する。

```text
1. 編成投稿を開始する
2. クエスト分類を選ぶ
   - 高難度攻略用
   - 周回・武器集め用
3. クエスト名を選ぶ
4. 属性を選ぶ
5. 利用可能なプリセットがあれば表示する
6. プリセットを選択した場合、フォームに初期値を反映する
7. 必要な項目だけ修正する
8. 投稿する
```

プリセットを使わず、空のフォームから投稿することもできるようにする。

---

## 5. 共通入力項目

すべての投稿で共通して扱う項目は以下。

| 項目 | 入力方式 | 必須 | 備考 |
|---|---|---:|---|
| 編成タイトル | 自由入力 | 必須 | 一覧カード・詳細画面で表示する |
| クエスト分類 | 選択式 | 必須 | 高難度攻略用 / 周回・武器集め用 |
| クエスト名 | 選択式＋自由入力 | 必須 | 候補にない場合は自由入力可能 |
| 属性 | 選択式 | 必須 | 火・水・土・風・光・闇 |
| 目的 | 選択式 | 必須 | クエスト分類により候補を切り替える |
| 操作タイプ | 選択式 | 必須 | 手動 / フルオート / セミオート / 未指定 |
| 検証状態 | 選択式 | 必須 | 未検証 / 投稿者クリア済 / 団内クリア済 / 要調整 |
| 概要メモ | 自由入力 | 任意 | 編成の特徴・用途を記載 |

---

## 6. 目的の選択肢

### 高難度攻略用

```text
団内挑戦
団内練習
個人練習
ソロ挑戦
クリア編成
参考メモ
```

### 周回・武器集め用

```text
周回向け
自発向け
救援向け
青箱狙い
参考メモ
```

---

## 7. 操作タイプ

操作タイプは目的とは別項目として扱う。

```text
手動
フルオート
セミオート
未指定
```

例：

```text
ルシゼロ：団内練習 × 手動
天元フリクエ：個人練習 × 手動
ムゲンHL：自発向け × フルオート
シエテHL：救援向け × 手動
```

---

## 8. 検証状態

検証状態は以下を基本とする。

```text
未検証
投稿者クリア済
団内クリア済
要調整
```

| 状態 | 意味 |
|---|---|
| 未検証 | 参考として登録しただけ |
| 投稿者クリア済 | 投稿者がこの編成でクリア済 |
| 団内クリア済 | 団内で実際に使ってクリア済 |
| 要調整 | 現環境や手持ちでは調整が必要 |

---

## 9. 高難度攻略用の追加項目

高難度攻略用では、以下を追加項目として扱う。

| 項目 | 入力方式 | 必須 | 備考 |
|---|---|---:|---|
| 役割 | 選択式＋自由入力 | 任意 | 火力、弱体、回復、予兆解除など |
| 予兆対応 | 自由入力 | 任意 | どの予兆をどう解除するか |
| 行動メモ | 自由入力 | 任意 | HP帯ごとの動き |
| 失敗ポイント | 自由入力 | 任意 | よく事故る箇所 |
| 必須パーツ | 選択式＋自由入力 | 任意 | キャラ・石・武器を指定 |
| 代用候補 | 自由入力 | 任意 | 代用できるパーツ・注意点 |
| 参考URL | URL入力 | 任意 | 攻略記事、YouTubeなど |

行動メモには、必要に応じて以下の見出しを使えるようにする。

```text
開幕
100〜80%
80〜60%
60〜20%
20〜0%
予兆対応
失敗ポイント
その他メモ
```

すべての見出しを必須入力にはしない。

---

## 10. 周回・武器集め用の追加項目

周回・武器集め用では、以下を追加項目として扱う。

| 項目 | 入力方式 | 必須 | 備考 |
|---|---|---:|---|
| 周回目的 | 選択式 | 任意 | 自発、救援、青箱狙いなど |
| 青箱狙い | 選択式 | 任意 | あり / なし / 未指定 |
| 討伐時間目安 | 自由入力 | 任意 | 例：3分、5分、10分など |
| 安定度 | 選択式 | 任意 | 安定、たまに失敗、要手動確認など |
| 編成前提 | 選択式 | 任意 | マグナ、神石、どちらでも、未指定 |
| 武器集め対象 | 自由入力 | 任意 | 滅尽剣、シュレディンガーなど |
| 救援タイミングメモ | 自由入力 | 任意 | 自発時の救援出しタイミングなど |
| 周回時の注意点 | 自由入力 | 任意 | 事故ポイントや確認事項 |

---

## 11. 編成パーツ入力

編成パーツとして以下を扱う。

```text
主人公ジョブ
キャラ
召喚石
武器
```

基本は候補リストから選択する。候補にないものは自由入力できる。

---

## 12. パーツ重要度

各パーツには重要度を設定できるようにする。

```text
必須
推奨
代用可
自由枠
```

| 重要度 | 意味 |
|---|---|
| 必須 | 代用が難しい |
| 推奨 | あると安定・速度が上がる |
| 代用可 | 他候補でも成立する |
| 自由枠 | 手持ちや好みに応じて変更可能 |

---

## 13. キャラ入力

キャラについては以下を扱う。

| 項目 | 内容 |
|---|---|
| 配置 | フロント / サブ / 任意 |
| キャラ名 | 選択式＋自由入力 |
| 重要度 | 必須 / 推奨 / 代用可 / 自由枠 |
| 役割メモ | 予兆解除、回復、火力、弱体など |
| 代用メモ | 代用候補があれば記載 |

---

## 14. 召喚石入力

召喚石については以下を扱う。

| 項目 | 内容 |
|---|---|
| 配置 | メイン / フレンド / サブ / 任意 |
| 召喚石名 | 選択式＋自由入力 |
| 重要度 | 必須 / 推奨 / 代用可 / 自由枠 |
| 用途メモ | 召喚効果、火力、耐久、予兆解除など |
| 代用メモ | 代用候補があれば記載 |

---

## 15. 武器入力

武器は最初から完全な10枠再現を必須にしない。

MVPでは、重要武器・必須武器・代用しづらい武器を中心に登録できればよい。

| 項目 | 内容 |
|---|---|
| 武器名 | 選択式＋自由入力 |
| 重要度 | 必須 / 推奨 / 代用可 / 自由枠 |
| 本数 | 1本、2本など |
| 用途メモ | 上限、耐久、奥義、アビ軸など |
| 代用メモ | 代用候補があれば記載 |

---

## 16. 参考URL

参考URLは複数登録できるようにする。

URL種別は以下。

```text
攻略記事
YouTube
X
note / ブログ
その他
```

URLごとの入力項目は以下。

| 項目 | 内容 |
|---|---|
| URL | 参考リンク |
| 種別 | 攻略記事、YouTubeなど |
| タイトル | 任意 |
| メモ | どこを見ればよいか |

MVPでは、URL登録できることを優先する。YouTubeの画面内再生はMVP対象外とする。

---

## 17. MVPに含めるもの

```text
共通項目の入力
高難度攻略用の追加項目
周回・武器集め用の追加項目
プリセットからの初期値反映
反映後の編集
キャラ・召喚石・武器の候補選択
候補にないものの自由入力
必須 / 推奨 / 代用可 / 自由枠の設定
参考URL登録
空フォームからの投稿
既存投稿をコピーして新規投稿
```

---

## 18. MVPではやらないもの

```text
ダメージ計算
最適編成の自動算出
ゲーム内データの自動取得
全武器10枠の完全再現必須化
YouTubeの画面内再生
所持状況との自動照合
外部サイトからの自動プリセット生成
```

---

## 19. 完了条件

以下ができれば、この機能のMVPは完了とする。

```text
ユーザーが高難度攻略用または周回・武器集め用の編成を投稿できる
プリセットを選んだ場合、フォームに初期値が反映される
反映後に必要な項目だけ修正できる
キャラ・召喚石・武器を候補リストまたは自由入力で登録できる
必須 / 推奨 / 代用可 / 自由枠を設定できる
参考URLを登録できる
空フォームからも投稿できる
既存投稿をコピーして新規投稿できる
```
