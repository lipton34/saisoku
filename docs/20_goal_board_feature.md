# 目標ボード・優先度管理 Phase 2 実装メモ

## 目的

既存の団内目標共有機能を拡張し、初心者・復帰者が「今なにをすればよいか」を整理しやすい目標ボードを追加する。

既存の目標一覧、目標作成、目標提案、届いた提案は維持する。

## 対象外

- ドラッグ＆ドロップによる列移動
- 目標と編成・武器・討伐数の実データ連結
- サブタスクの実テーブル化
- 素材数の詳細管理
- 自動最適化や所持状況の自動判定

## DB変更

`shared_goals` に以下のカラムを追加した。

- `boardStatus`
- `priority`
- `effort`
- `beginnerRecommended`
- `sortOrder`

`dueDate` は既存カラムを利用する。

`status` は従来通り `達成` / `未達成` を表す。
ボード上の列は `boardStatus` で管理する。

## API変更

`/api/shared-goals` の作成・更新・一覧取得で Phase 2 の項目を扱えるようにした。

一覧取得では以下の絞り込みにも対応した。

- `boardStatus`
- `priority`
- `effort`
- `beginnerOnly`

## 画面変更

`/goals` に「目標ボード」タブを追加した。

ボード列:

- 今やる
- 次にやる
- 後でやる
- 保留
- 完了

カード表示:

- 目標タイトル
- カテゴリ
- 優先度
- 負荷
- 期限
- 初心者向けラベル
- 進捗
- 必要武器数

状態変更はドラッグ＆ドロップではなく、カード内のセレクトで行う。

## 目標詳細

目標詳細に以下を追加した。

- ボード状態
- 優先度
- 負荷
- 初心者向け
- Phase 3予定枠
  - 関連編成
  - 必要武器
  - 討伐目標
  - サブタスク

## 今後の拡張余地

- ドラッグ＆ドロップ移動
- 関連編成の実リンク
- 必要武器と本数の管理
- 討伐目標の管理
- サブタスク追加・完了チェック
