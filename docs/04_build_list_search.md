# Codex作業指示：04_編成一覧・検索画面

## 1. 目的

`saisoku` に、高難易度向け編成共有機能の一部として **編成一覧・検索画面** を追加・整備する。

この画面の目的は、団員が目的の編成をすぐ見つけられるようにすることである。

投稿された編成やプリセットが増えても、クエスト・属性・目的・操作タイプ・検証状態・キャラ名などで絞り込み、必要な編成に素早く到達できるようにする。

## 3. 基本方針

編成一覧・検索画面は、単なる投稿一覧ではなく、**団内の編成検索・参照ハブ**として扱う。

重視することは以下。

```text
自分が挑戦したいクエストの編成をすぐ探せる
自分の属性に合う編成を探せる
手動・フルオート・青箱狙いなどの目的で絞り込める
検証済みの編成を優先して見つけられる
必要キャラ・召喚石・武器で検索できる
古い編成や要調整の編成を判別できる
```

---

## 4. 画面上部の分類切り替え

一覧画面の上部では、まず以下を切り替えられるようにする。

```text
すべて
高難度攻略用
周回・武器集め用
```

この切り替えによって、ルシゼロ・天元系を探したい人と、レヴァンス周回編成を探したい人を分ける。

---

## 5. 共通フィルター

全分類共通で使うフィルターは以下。

| フィルター | 内容 |
|---|---|
| クエスト分類 | 高難度攻略用 / 周回・武器集め用 / すべて |
| クエスト名 | ルシゼロ、天元HL、ムゲンHLなど |
| 属性 | 火・水・土・風・光・闇 |
| 目的 | 団内練習、個人練習、周回、自発、救援など |
| 操作タイプ | 手動、フルオート、セミオート、未指定 |
| 検証状態 | 未検証、投稿者クリア済、団内クリア済、要調整 |
| 投稿種別 | 投稿編成 / プリセット / すべて |
| キーワード | タイトル、メモ、キャラ名、石名、武器名など |

## 9. 一覧カードの役割

一覧カードは、**詳細を見るかどうかを判断するための画面要素** とする。

詳細画面で情報を重複させすぎないため、カードでは必要な判断情報を簡潔に表示する。

---

## 10. 一覧カードで表示する情報

一覧カードでは以下を表示する。

| 表示項目 | 内容 |
|---|---|
| 編成タイトル | 投稿名 |
| 属性 | 火・水・土・風・光・闇 |
| 目的 | 団内練習、周回、救援など |
| 操作タイプ | 手動、フルオートなど |
| 検証状態 | 未検証、投稿者クリア済など |
| 投稿種別 | プリセット / 投稿編成 |
| 投稿者 | 誰の投稿か |
| 更新日 | 最終更新日 |
| 主要パーツ | 主なキャラ・石・武器 |
| 参考URL有無 | 動画あり、記事ありなど |
| 行動メモ有無 | メモあり / なし |

必要に応じてクエスト名も表示する。ただし、詳細画面では重複表示を抑える。

---

## 11. 一覧カードの表示イメージ

```text
ルシゼロ 水剣豪 初挑戦向け

[水] [手動] [団内練習] [投稿者クリア済]
主要パーツ：剣豪 / キャラA / キャラB / 召喚石C
参考：YouTubeあり / 攻略記事あり
行動メモ：あり
更新：2026/05/14
```

周回・武器集め用の例：

```text
ムゲンHL 火フルオート自発向け

[火] [フルオート] [自発向け] [安定] [神石]
主要パーツ：キャラA / 召喚石B / 武器C
参考：記事あり
周回メモ：あり
更新：2026/05/14
```

---

## 12. プリセットと投稿編成の区別

一覧上では、プリセットと投稿編成を区別できるようにする。

| 種別 | 意味 |
|---|---|
| プリセット | 団内テンプレート。投稿の初期値として使える |
| 投稿編成 | 団員が実際に投稿した編成 |

プリセットは投稿開始時に選べるだけでなく、一覧からも確認できると便利である。

## 14. クイックフィルター

通常の詳細フィルターに加え、よく使う条件をワンタップで切り替えられると便利である。

候補：

```text
ルシゼロ
天元HL
レヴァンス
自発向け
救援向け
フルオート
団内クリア済
```

MVPで余裕があれば実装する。必須ではない。

## 15. お気に入り・ピン留め

以下は将来検討とする。

```text

```

```text

```
